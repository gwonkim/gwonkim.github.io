# string : 문자열
> 자바스크립트에서의 문자열은 기본 자료형.
> 자바에서는 복합 자료형이지만 자바스크립트의 문자열은 기본 자료형이기 때문에 주의헤야 한다.

## String Method
> 불변성 문자열(Immutable String)
>> 원형을 변형할 수 없음. 원본 유지!
>> 복사되어서 변형된다는 점을 유의!
> 리턴 값이 false일 경우 -1 리턴
