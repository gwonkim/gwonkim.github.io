EC2 진입

  • SSH 연결
    : ssh -i keyOrPemFile ubuntu@퍼블릭IP
    : ssh -i keyOrPemFile ubuntu@도메인네임
  • 연결 종료
    : exit

패키지 관련 명령(데몬)

    패키지 업데이트

  • 설치된 패키지 업데이트 항목 다운 : sudo apt update
    설치된 패키지 업그레이드 : sudo apt upgrade

    • 패키지 설치 및 상태 변경

  • 설치 : sudo apt install 패키지명
    삭제 : sudo apt remove 패키지명
    상태 확인 : service apt status 패키지명
    종료 : sudo service apt stop 패키지명
    재시작 : sudo service apt restart 패키지명

파일 다운로드 및 업로드

  • 다운로드
    : EC2 서버에 있는 파일을 내 PC로 다운로드
    : scp -i 키파일.pem ubuntu@IP주소:우툰부파일위치주소 내PC주소
    예시) scp -i 키파일.pem ubuntu@IP주소:~/department.sql.gz ~
    (우분투EC2 서버의 ~/department.sql.gz를 내 PC의 홈디렉토리(~)로 다운로드)
    : 현재 디렉토리 다운로드하고 싶다면 . 문자 사용
    : 문서 폴더로 다운로드하고 싶다면 ~/Documents 사용
  • 업로드
    : 내 PC에 있는 파일을 EC2 서버로 다운로드
    : scp -i 키파일.pem 내pc파일위치주소 ubuntu@IP주소:~
    예시) scp -i 키파일.pem ~/department.sql.gz ubuntu@IP주소:~
    (내 pc의 홈디렉토리(~)에 있는 department.sql.gz를 우분투 EC2서버의 홈디렉토리(~)로 다운로드)
    : 현재 디렉토리 다운로드하고 싶다면 . 문자 사용
    : 문서 폴더로 다운로드하고 싶다면 ~/Documents 사용
