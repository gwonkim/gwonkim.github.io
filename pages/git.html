GIT

Git이란?

  • 소스코드 파일의 버전 백업(코드 버전 관리)
  • 프로젝트 공동 작업(협업)
  • 리눅스 토발즈가 개발
    • => 소스 관리를 위한 분산 버전 관리 시스템

CLI vs GUI

  • CLI란?
    CLI: Command Line Interface
    -> cmd/terminal/shell에서 명령을 입력하는 사용자 인터페이스
  • GUI란?
    GUI: Graphic User Interface
    -> 메뉴나 버튼을 클릭하여 명령을 입력하는 사용자 인터페이스

☞ Git 세부 설명 ☜

  • State 클릭하면 State페이지로 이동합니다.
  • Repository 클릭하면 Repository페이지로 이동합니다.
  • Merge 클릭하면 Merge페이지로 이동합니다.
  • Git Page 클릭하면 Git Page페이지로 이동합니다.
  • branch 클릭하면 branch페이지로 이동합니다.
  • commit 클릭하면 commit페이지로 이동합니다.
  • Command 클릭하면 Command페이지로 이동합니다.
  • aws의 우분투 접속 클릭하면 aws페이지로 이동합니다.
