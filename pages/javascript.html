JavaScript

‖> javascript란?

    자바(Java)랑 헷갈리지 말아야 한다.

    js는 멤버 변수, 메소드 다 속성

    java는 멤버 변수 X, 메소드 O


‖> 자료형

  • 기본 자료형

    • string, number, boolean, undefined

  • 복합 자료형

    • function, object


‖> 문자열 : String


‖> 배열 : Array


‖> 함수형 프로그래밍 : Function


‖> 엄격한 문법 사용 : use strict

    js 소스 코드 파일 선두에 "use strict"를 선언

    이 소스 파일은 변수를 미리 선언하지 않고 사용할 경우 문법 오류 발생


‖> 호이스팅 : Hoisting


‖> 모듈 사용 : import VS require
