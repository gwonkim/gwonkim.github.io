# ts와 tsx

## .ts 
- Typescript - .ts
- 순수 TypeScript 파일에 사용 합니다.
- JSX 요소 추가를 지원하지 않습니다.

## .tsx 
- React Component - .tsx
- JSX를 포함하는 파일에 사용 합니다.
