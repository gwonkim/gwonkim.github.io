# Git : Git Page
- 레포지토리 이름을 사용자이름.github.io로 만들어 md나 html 파일을 이용하여 하나의 사이트를 만들 수 있다.
            
## 참고사항
- html을 사용해 정리하는 경우만 해당
- git page를 작성할 때는 .md 파일을 만들 경우 .html이 작동을 안 함. 이중으로 만들면 안 된다.
- 깃 페이지를 작성할 때 live Server를 이용하면 빠르게 변경사항을 확인할 수 있다.
    - liveServer 설치 후 명령어 live-server
