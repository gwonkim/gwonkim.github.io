polyfill이란?

  • 지원하지 않는 이전 브라우저에서 최신 기능을 제공하는 데 필요한 코드
  • 계속 변화하는 자바스크립트와 이전 버전을 호환하기 위해서 생긴 것
  • 최신 자바스크립트로 기능을 구현한 경우 호환이 안 되는 브라우저가 있을 수 있기 때문에 이전 버전의 코드로 똑같이 구현한 코드
  • polyfill의 뜻 = 충전솜 변경된 표준을 준수할 수 있게 기존 함수의 동작 방식을 수정/새롭게 구현한 함수의 스크립트 : 폴리필(polyfill) -> 폴리필(polyfill)은 말 그대로 구현이 누락된 새로운 기능을 메꿔주는(fill in) 역할

babel이란?

  • 트랜스파일러(transplier)로 모던 JS 코드로 구 표준을 지키는 코드로 변환
  • 바벨은 코드를 재작성해주는 트랜스파일러 프로그램
