React

Start

  1. 설치 및 버전 확인
    node -v
    npm -v
  2. npm 초기화 설정
    npm init

    3. * 특정 버전을 설치하고 싶다면 @ 문자 사용(ex: npm i next@8, next 8버전 설치)

  3. dependencies(의존성) 설치
    npm i next npm i react npm i react-dom npm i prop-type
  4. devDependencies(개발 의존성) 설치
    npm i -D eslint eslint-plugin-import eslint-plugin-react eslint-plugin-react-hooks nodemon webpack
